title: Як створити контейнер зі збреженням файлів?
tags:
  - question
  - docker
aliases:
---
```bash
docker volume create my-volume
docker run -it --rm my-volum:/my-data ubuntu:22.04
```
or
```
docker run -it --rm --mount source=my-volume,destination=/my-data/ ubuntu:22.04
```
