> [!tldr]
> **Dockerfile** — текстовий файл, який містить команди, які користувач може викликати у командному рядку для створення образу.

> [!example] Recipe
> 1. Start with OS
> 2. Install language runtime
> 3. Install application deps
> 4. Set up execution environment
> 5. Run application
